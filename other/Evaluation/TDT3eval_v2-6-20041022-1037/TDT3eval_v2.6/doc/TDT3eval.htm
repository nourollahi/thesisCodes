August 12, 1999

This HTML manual set serves as the user manual for NIST's TDT Phase 2 evaluation package. The user interface consists of four PERL5 programs, TDT3seg.pl, TDT3trk.pl, TDT3det.pl, TDT3fsd.pl, and DetectionScore.pl corresponding to the segmentation, tracking, topic detection, first story detection, and story link detection tasks respectively.

A TDT3eval FAQ has been exists to help explain common usage practices.

There are a number of supporting scripts available in support if the TDT3 Evaluation:

TDT3BuildIndex.pl Generates TDT3 Evaluation indexes per the test specification

The Installation instructions are in the ASCII text file readme.txt which is located in the main directory of this distribution. This manual assumes that you have already successfully installed and tested the required program elements described in the readme.

The Revision History is maintained in doc/revision.htm.
